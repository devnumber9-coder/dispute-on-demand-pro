2. How Your Credit Data Is Processed

Local Processing Architecture

Your credit report data is processed 100% locally in your web browser using JavaScript. Here's exactly how it works:

  1. You upload or paste your credit report
  2. JavaScript in your browser parses the text locally
  3. Letters are generated using client-side code
  4. PDFs are created in your browser (html2pdf.js library)
  5. Nothing is transmitted to our servers
  6. When you close the page, all data is cleared from browser memory

Technical verification: You can verify this by:

  • Opening browser Developer Tools (F12) ‚Üí Network tab
  • Uploading a credit report and generating letters
  • You'll see ZERO network requests containing your credit data
